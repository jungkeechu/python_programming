What you will learn in Python

  • programming environment: web compiler
    1. 우리는 지금부터 구글(Google)이 제공하는 Colab이라는 환경을 사용할 것이다. Colab은 주피터 노트북(Jupyter Notebook)이라는 파이썬 개발 환경인데, 별도로 설치할 필요가 없어 편하다. 아래의 사이트에 접속하자. 구글 ID로 로그인할 수 있다. [Google Colab]
    2. 왼쪽 상단의 메뉴에서 [File]—[New Python 3 Notebook]을 클릭해서 새로운 파일을 생성하자. 파이썬 언어도 버전을 업그레이드하고 있는데 ‘Python 3’버전을 사용하려는 것이다. Colab에서는 Notebook이라는 단위가 파일이라고 생각하면 된다. 이렇게 생성된 파일은 본인의 Google 계정의 구글 드라이브 폴더에 저장된다([Google Drive]).
    3. 화면의 작은 박스를 셀(cell)이라고 하는데 셀 단위로 프로그램하고, 셀 단위로 프로그램을 실행할 수 있다
      • (실행 방법: 1)셀의 좌측의 실행 버튼 클릭 또는
      • (실행 방법: 2) Control+Enter키.
    4. [샘플 사이트]를 접속해보자. 이 페이지는 저자가 만들어 둔 페이지라서 실행하려면 fork 해서 사용해야한다.
    5. 코드 파일을 실행해보고, [File]—[Save] 메뉴에서 저장하면, https://drive.google.com 사이트에 [Colab Noteboos]라는 폴더 내부에 파일이 저장된다. 각자 확인해보자.
